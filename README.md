## ColorTherapy_IoT

+ #### H/W & S/W
  + H/W
    + Arduino UNO
    + DHT11 Sensor
    + RGB LED
    + Liquid Crystal Display(I2C)
  + S/W
    + Arduino IDE
    + Node.js
    + Express framework
    + PUG 
    + Mongoose DB
+ #### Structure 
   + webserver ↔️ Arduino : Serial Port
   + webserver ↔️ website : http protocol 

+ #### npm Install
```
$ npm install -g nodemon
$ npm i express pug mongoose serialport @serialport/parser-readline
```
